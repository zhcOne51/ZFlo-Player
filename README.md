## 产品简介

**ZFlo Player** 是一款专为Windows用户打造的现代化视频播放器与媒体管理中心。支持全格式视频播放、智能拼音搜索、收藏列表管理，为您提供流畅的播放体验和高效的视频库管理方案。

### 核心优势

- **简单易用** - 拖入文件夹即可开始，无需复杂配置
- **智能搜索** - 支持拼音输入快速定位中文视频
- **全能播放** - 兼容MP4/MKV/AVI等主流格式及HLS/DASH网络流
- **强大组织** - 收藏夹、播放列表、标签系统一应俱全

## 核心功能

### 视频播放

- **全格式支持**：MP4, MKV, AVI, MOV, WMV, FLV, TS/M2TS等
- **硬件加速**：充分利用显卡性能，流畅省电
- **画中画模式**：边看视频边处理其他工作
- **一键截图**：按S键保存当前画面
- **字幕/音轨**：自动识别多语言字幕和音轨切换
- **进度记忆**：关闭后再次打开自动跳转到上次位置

| 快捷键 | 功能 |
|:-------|:-----|
| `空格` | 播放/暂停 |
| `←` / `→` | 快退/快进10秒 |
| `F` | 全屏 |
| `M` | 静音 |
| `P` | 画中画 |
| `S` | 截图 |

### 媒体库管理

- **文件夹管理**：拖放导入、树形浏览、实时监控文件变化
- **拼音搜索**：输入拼音首字母或全拼即可匹配中文视频名
- **高级筛选**：按格式、分辨率、时长、标签等多维度过滤
- **收藏系统**：一键收藏喜爱的视频，独立视图查看
- **播放列表**：自定义观影顺序，拖拽调整顺序
- **继续观看**：首页展示未看完的视频，带进度条提示

### 网络视频

- URL直链播放（MP4/MKV）
- HLS流媒体（.m3u8）
- DASH自适应流（.mpd）
- 剪贴板智能检测视频链接

### 界面定制

- **双主题**：亮色/暗色自由切换
- **双视图**：网格视图 / 列表视图
- **可调侧边栏**：自由调节宽度或完全收起

## 快速上手

### 添加视频

1. 点击左侧边栏底部 **「+ 添加文件夹」**
2. 或直接将文件夹**拖入**窗口
3. 视频 自动扫描入库并生成缩略图

### 搜索视频

点击顶部搜索框，输入关键词（支持中文、英文、**拼音**），结果实时过滤。

### 组织管理

- **收藏**：右键视频 → 「添加到收藏」
- **播放列表**：「播放列表」→「+ 创建」→ 拖入视频
- **排序/筛选**：使用列表上方的工具栏

## 常见问题

**Q: 视频无法播放？**

A: 尝试启用「智能软解码」（设置 → 播放器）；确认文件未损坏；更新到最新版本。

**Q: 字幕不显示？**

A: 确保字幕文件(.srt/.ass)与视频同名且在同目录下；在播放器中选择正确的字幕轨道。

**Q: 缩略图很慢？**

A: 首次扫描需要时间，后台会自动生成；建议使用SSD存放视频以提升速度。

**Q: 如何备份数据？**

A: 备份整个配置目录：`C:\Users\您的用户名\AppData\Roaming\zflo-player\`
